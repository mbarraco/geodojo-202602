# geodojo-202602
Repositorio de contenidos para las clases del curso inical de Python en GeoDojo,


# Programa
1. Introducción a Python y entorno de trabajo: tipos de datos, operadores, funciones básicas.

2. Estructuras de datos y control de flujo: listas, tuplas, diccionarios, condicionales y bucles.

3. Archivos, módulos y buenas prácticas: lectura/escritura CSV, manejo de errores, estilo PEP8.

# Ejercitación

En el directorio `ejercicios` encontrarás 1 archivo correspondiente a cada unidad. Cada archivo incluye ejericios en tres categorías: A, B y C. Cada categoría corresponde a una propuesta de trabajo distinta, a saber:
- A: ejercicios fundamentales, necesarios y (probablemente) suficientes para entender el tema. Se recomienda hacerlos todos
- B: ejercicios para quien quiera practicar un poco más. No están pensados como estrictamente necesarios, pero es recomendable hacerlos.
- C: ejercicios que requieren cierto grado de dificultad extra. Podrían requerir investigación independiente sobre la materia, o la combinación de varias técnicas ya aprendidas. Son opcionales
